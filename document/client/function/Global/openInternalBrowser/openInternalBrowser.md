# openInternalBrowser

## Input / Parameter

| Name | Description | Input Type | Default | Options | Required |
| ------ | ------ | ------ | ------ | ------ | ------ |
| url | The url to open within the page. | String/Text | - | - | Yes |

N/A

Note: The page will be updated to show the url passed by the user.
