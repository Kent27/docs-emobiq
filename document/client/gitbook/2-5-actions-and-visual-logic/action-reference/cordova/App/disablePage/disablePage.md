# disablePage

## Description

Disable the ability to click on the application.

### Platform Supported

- Mobile
- Web

## Input / Parameter

N/A

## Output

| Description | Output Type |
| ------ | ------ |
| Returns 'true'. | Boolean |

Note: The application would be disabled, which means you won't be able to click anywhere in the app.
