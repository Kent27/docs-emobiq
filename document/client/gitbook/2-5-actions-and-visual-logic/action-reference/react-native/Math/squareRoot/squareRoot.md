# Math.squareRoot

## Description

Get the square root of a number.

## Input / Parameter

| Name | Description | Input Type | Default | Options | Required |
| ------ | ------ | ------ | ------ | ------ | ------ |
| value | The number to return the square root of. | Number | - | - | Yes |

## Output

| Description | Output Type |
| ------ | ------ |
| Returns the square root value. | Number |
