# enablePage

## Description

Enable the ability to click on the application.

### Platform Supported

- Mobile
- Web

## Input / Parameter

N/A

## Output

| Description | Output Type |
| ------ | ------ |
| Returns 'true'. | Boolean |

Note: The application would be enabled, which means you will be able to click anywhere in the app.
