# sc_esc

## Description

Returns an ASCII escape character, which refers to the table below:

| Decimal | Hex | Oct | Char |
| ------ | ------ | ------ | ------ |
| 27 | 1B | 033 | ESC |

## Input / Parameter

N/A

## Output

| Description | Output Type |
| ------ | ------ |
| Returns the ASCII escape character. | String/Text |
