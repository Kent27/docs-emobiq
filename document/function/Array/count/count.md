# Count

## Description

Count the number of elements in an array.

## Input / Parameter

| Name | Description | Input Type | Default | Options | Required |
| ------ | ------ | ------ | ------ | ------ | ------ |
| values | The array of values. | Array | - | - | Yes |

## Output

### Default Output

| Description | Output Type |
| ------ | ------ |
| The total number of elements in the array. | Number |
