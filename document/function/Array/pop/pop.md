# Pop

## Description

Delete the last value inside an array.

## Input / Parameter

| Name | Description | Input Type | Default | Options | Required |
| ------ | ------ | ------ | ------ | ------ | ------ |
| var | The name of the array. | String | - | - | Yes |

## Output

### Default Output

| Description | Output Type |
| ------ | ------ |
| The updated array of values. | Object |

```js
    {
        "filePath": "{filePath}"
    }
```
